Privacy Policy

Overview

The Open Braille Beacon website (lamdevelopper.github.io/Open-Braille_beacon) does not collect, store, process, or sell any personal data from visitors.

Data we do not collect

  • No analytics or tracking scripts (Google Analytics, cookies, or similar)
  • No IP address logging beyond standard server access logs managed by GitHub Pages
  • No cookies or local storage
  • No user accounts, forms, or input fields
  • No device identifiers or BLE signal data — this website describes the concept only

Third-party services

This site loads fonts from Google Fonts (fonts.googleapis.com, fonts.gstatic.com). Google may collect your IP address to serve the font files. See Google's Privacy Policy.

Links to GitHub (github.com) redirect you to a third-party site. GitHub has its own privacy policy: GitHub Privacy Statement.

GitHub Pages hosting

This site is hosted on GitHub Pages. GitHub may collect standard server access logs (IP address, browser type, requested URL, timestamp) as part of infrastructure operations. This data is not accessible to the project maintainers. GitHub Pages does not set cookies to visitors of this site. See GitHub Privacy Statement.

Changes to this policy

If we add analytics, cookies, or any data-collecting feature in the future, this policy will be updated before the feature goes live and the change will be noted in the repository.

Contact

Questions about this policy can be raised through GitHub Issues.
